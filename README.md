
# opencv2-python

## Pasos recomendados
Para usar la app se recomienda usar la version de python 3.12.4.
- Link de instalador de python para windows: [Python 3.12.4](https://www.python.org/ftp/python/3.12.4/python-3.12.4-amd64.exe)
- Para otras plataformas: [Python 3.12.3
](python.org/downloads/release/python-3124/)
- Actualizar pip
```bash
# pip install --upgrade pip
python.exe -m pip install --upgrade pip
```
- Instalar las librerias necesarias
```bash
pip install -r requirements.txt
```
- Ejecutar el archivo main.py de la carpeta app
```bash
cd app
python main.py 
# o directamente python ./app/main.py
```
